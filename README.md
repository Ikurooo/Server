## Installation

git clone this repo

## Instructions

use make to build the server

## Usage

```sh
./server [-p PORT] [-i INDEX FILE] DOCROOT
```

## Features
- Handle GET requests
- Capable of responding with: 200 400 403 404 418 500 501
- Every incoming request gets printed to stderr
